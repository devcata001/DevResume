# Quick Start Guide - DevResume

Get your professional resume ready in 5 minutes! üöÄ

## Option 1: Start with a Profession Preset (Recommended)

1. **Open** `index.html` in your browser
2. **Select** your profession from the dropdown:
   - Frontend Developer
   - Backend Developer
   - Full Stack Developer
   - Application Security Engineer
   - DevOps Engineer
   - Data Engineer
3. **Click** to load sample data
4. **Edit** the pre-filled information with your details
5. **Choose** your favorite template (Minimal, Modern, or Classic)
6. **Export** to PDF when ready!

## Option 2: Start from Scratch

1. **Open** `index.html` in your browser
2. **Fill in** your information:
   - Personal details (name, email, phone, etc.)
   - Professional summary
   - Skills (press Enter to add each skill)
   - Work experience (click "+ Add" for each position)
   - Education
   - Projects
   - Certifications
3. **Pick** a template from the dropdown
4. **Export** to PDF

## Keyboard Shortcuts

- **Ctrl/Cmd + P**: Export to PDF
- **Ctrl/Cmd + S**: Force save (already autosaves)
- **Ctrl/Cmd + E**: Export JSON backup
- **Enter** (in skills field): Add skill tag

## Export Your Resume

### As PDF (Recommended)

1. Click **"Export PDF"** button
2. In print dialog:
   - Destination: "Save as PDF"
   - Layout: Portrait
   - Background graphics: ‚úÖ Enabled
   - Margins: None or Minimum
3. Click **Save**

### As JSON (Backup)

1. Click **"Export JSON"** button
2. File downloads automatically
3. Save somewhere safe
4. Import later to restore

## Working Offline

DevResume works 100% offline!

- No internet required after initial load
- All data stored in your browser
- Export anytime without connection

## Need Help?

- **Resume not saving?** Check you're not in private/incognito mode
- **PDF looks wrong?** Use Chrome/Edge and enable background graphics
- **Template not changing?** Hard refresh (Ctrl+Shift+R)
- **Want to start over?** Click "Clear All" button

## Example Workflow

**Scenario**: You're a Frontend Developer applying for a new job

1. Open DevResume
2. Select "Frontend Developer" from profession dropdown
3. Confirm to load sample data
4. Replace sample name/email with yours
5. Edit work experience with your actual roles
6. Add your real skills (press Enter after each)
7. Remove sample projects, add yours
8. Switch to "Modern" template (looks great for tech!)
9. Export to PDF
